# DevOps Lab 🚀

Laboratório para estudos e crises do DevOps.

## 📚 Projetos

### 1. Server Stats Script
Script em Bash para monitorar estatísticas de desempenho de servidores Linux.

📂 **Localização:** `/server-stats`

---

### 2. Desafio DevOps - Lacrei Saúde 💚
Pipeline CI/CD completo para deploy automatizado de uma aplicação Node.js na AWS, com ambientes isolados de staging e produção (EC2 + Docker + Nginx + GitHub Actions), monitoramento via CloudWatch e 4 estratégias de rollback documentadas e testadas.

📂 **Localização:** `/Desafio-DevOps-Lacrei-Sa-de`
📄 **Documentação:** [README do projeto](./Desafio-DevOps-Lacrei-Sa-de/README.md)

---

### 3. Terraform AWS - EKS Cluster Lab 🌐
Infraestrutura como código para provisionar um cluster EKS na AWS usando Terraform, com estrutura e módulos separados (`network` e `cluster`) VPC, subnets públicas/privadas, NAT/Internet Gateway, IAM roles e o cluster EKS.

📂 **Localização:** `/TerraformAWS`


---

## 🛠️ Tecnologias Estudadas

- Bash Scripting
- Linux System Administration
- Monitoring & Observability (CloudWatch)
- Docker & Docker Compose
- Nginx (reverse proxy / SSL)
- CI/CD com GitHub Actions
- AWS (EC2, VPC, IAM, EKS)
- Terraform (módulos, providers, state)
